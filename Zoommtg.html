init(params)

Init Zoom Meeting
Parameters:
Name Type Description
params Object
Properties
Name Type Attributes Default Description
debug boolean <optional>
 false Turn on debug mode can print log in browser console log
leaveUrl string Post message page url after leave meeting, like “http://www.zoom.us”
showMeetingHeader boolean <optional>
 true Show or hide the meeting header, includes meeting number and topic
disableInvite boolean <optional>
 false Enable or disable invite function
disableCallOut boolean <optional>
 false Enable or disable call out function
disableRecord boolean <optional>
 false Enable or disable record function
disableJoinAudio boolean <optional>
 false Enable or disable join audio function
audioPanelAlwaysOpen boolean <optional>
 false Set the default state of the audio panel on join
success Zoommtg~successCallback Call back function on success
error Zoommtg~errorCallback Call back function on error

join(params)

Join Meeting
Example
Zoommtg.join({
	meetingNumber: 123456789,
	userName: 'Zoomie Userton',
	password: '',
	apiKey: 'API_KEY',
	signature: 'SIGNATURE',
	participantId: 'UUID'
 });
Parameters:
Name Type Description
params Object
Properties
Name Type Default Description
meetingNumber int Zoom meeting number
userName string Display name in meeting
passWord string '' Meeting password, if no password, set to ''
apiKey string REST API Key
signature string See Generating a signature for a How-to
participantId string set meeting participant ID
